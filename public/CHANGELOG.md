## v1.1.0

### 创作工作流统一

- **「文生图」+「图像编辑」合并为「新的创作」**:`/create` 单一路由,顶部图片栏空则走文生图,塞参考图则自动切到图像编辑,无需先选页面再上传
- 参考图三种添加方式:**paperclip 多选** / **拖拽到窗口任意位置** / **Ctrl+V 粘贴剪贴板图像**
- ChatGPT 风格 composer:磨砂玻璃输入框 + 行内缩略图条 + 内联模型选择 chip
- **GSAP 渐变背景漂流**:4 个 blob 随机游走 + 独立 opacity 脉动,无 yoyo

### 生成任务全局化

- 全新 `tasks` Pinia 单例 store:**切换页面、进设置都不丢失正在进行的生成**
- 侧边栏「**生成中**」分组实时跟踪进度(常驻显示,无任务时显示空态)
- 完成后自动转入「**最近生成**」 — 从磁盘缓存读取,**刷新 / 重启不丢**
- 任一 chip 点击 → 直接回到 `/create` 结果页(运行中显示进度,完成显示结果),可重新生成、edit-this-image、批量保存
- 状态机驱动:URL `?task=<id>` 同时控制视图模式 + sidebar 高亮

### 大图查看器(ImageLightbox)

- 滚轮缩放(以光标位置为锚点)
- 拖拽平移、±90° 旋转
- 关闭按钮 + ESC 退出
- 生成页 + 历史详情页共享同一组件

### 首页升级

- **「新的创作」+「更新内容」** 双卡同高布局
- 最近生成网格:6 张缩略图,大屏自动收为单行 4 张
- **更新内容卡**:状态图标方块(检查中 loader 旋转 / 可用 ↑ / 最新 ✓ / 错误 ☁︎×)
- 更新日志默认折叠,「**查看更新日志**」popover 弹出 — 卡片始终紧凑,日志再长也不顶高
- 启动一次自动检查,**切页不再重复请求**

### Footer「新版本可用」常驻提示

- 有更新时底栏长亮 pill,标注 `v当前 → v新版`
- 点击弹悬浮窗:完整 release notes + 一键「**立即更新**」
- 在任何页面都能直接升级,不必跳回首页

### 发版流程改进

- `release.yml` 自动从 `public/CHANGELOG.md` 注入 GitHub Release body
- **一份 source of truth**:Release page / Tauri updater `latest.json` / 应用内卡片三处自动同步
- 更新日志随包发布,**离线也能在首页看到当前版本的更新内容**

### 快捷键 / 命名

- `Ctrl+N` → 新的创作 / `Ctrl+S` → 设置
- 「文生图」/「图像编辑」→ **新的创作**
- 「生成历史」→ **创作历史**

### 杂项

- 模型选择下拉框只显示模型名称(去掉端点后缀)
- 设置页 → 关于:**捐赠区**(收款码)
- 图像编辑支持多张参考图同时上传
- 流式生成的 partial 图也可以点击放大
- ResultsView 加「edit-this-image」按钮,一键把结果回填到 `/create` 作为下一次编辑的输入
